## What is QuickHW?

QuickHW (Presto AI) is a native macOS menu bar app that uses AI to solve homework problems. Press a hotkey, capture any problem on your screen — math, physics, chemistry, code, anything — and get a detailed step-by-step solution in a floating overlay. No tab-switching. No copy-pasting.

## Features

- **Instant Screen Capture** — Press Cmd+Shift+X to capture any problem on screen
- **Step-by-Step Solutions** — Detailed breakdowns powered by Claude AI
- **Floating Overlay** — Solutions appear next to your work, no tab switching
- **Study Mode** — Learn concepts with explanations, similar examples, and quizzes
- **Auto Solve** — Detect and solve entire worksheets in parallel
- **Quick Prompt** — Ask any question about what's on your screen
- **All Subjects** — Math, physics, chemistry, CS, statistics, essays, and more
